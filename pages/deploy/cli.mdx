# Installation

You can check how to install the Wasmer CLI in the [official docs](/runtime/cli)

# Usage

## `wasmer login`

Before being able to deploy and manage Applications on the edge, you will need to log in.
Make sure you have already created an account.
If you try to execute a command that requires being logged in, you will be warned and the command will be aborted.

To proceed, run `wasmer login` follow the instructions.

## `wasmer logout`

If you want, you can log out at any time.
To proceed, run `wasmer logout`.

## `wasmer deploy`

`wasmer deploy PATH`
