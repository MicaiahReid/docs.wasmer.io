# Wasmer Runners

Wasmer has a pluggable runner architecture.
Currently, Wasmer supports the following runners:

- [WASIX](https://docs.rs/webc/latest/webc/metadata/annotations/constant.WASI_RUNNER_URI.html)
- [Emscripten](https://docs.rs/webc/latest/webc/metadata/annotations/constant.EMSCRIPTEN_RUNNER_URI.html)
- [WCGI](https://docs.rs/webc/latest/webc/metadata/annotations/constant.WCGI_RUNNER_URI.html)
