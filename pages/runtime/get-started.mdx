# Get Started with the Wasmer Runtime

This page will guide you through installing Wasmer, and running a published container in the Wasmer Registry.
The runtime can run any pacakge published to the Wasmer Registry.

<Steps>
 
### Install Wasmer
 
Click [here](/install) for instructions on how to install Wasmer.

### Run a package

Let's say you want to run the Wasmer package [`python/python`](https://wasmer.io/python/python) and execute the `-c "print(1+1)"` to print a result.

<Tabs items={['CLI', /*'JS', 'Python'*/]}>
  <Tab>
    ```bash copy
    wasmer run python/python -- -c "print(1+1)"
    ```
