# Get Started with the Wasmer Runtime

This page will guide you through installing Wasmer, and running a published container in the Wasmer Registry.
The runtime can run any pacakge published to the Wasmer Registry.

<Steps>
 
### Install Wasmer
 
Click [here](/install) for instructions on how to install Wasmer.

<Tabs items={['CLI', 'JS', 'Python']}>
  <Tab>
    Unix (macOS, Linux):

    ```bash
    curl https://get.wasmer.io -sSfL | sh
    ```

    PowerShell (Windows):

    ```bash
    iwr https://win.wasmer.io -useb | iex
    ```

    <Callout type="info">
        Other installation methods available here: https://github.com/wasmerio/wasmer-install
        </Callout>

  </Tab>
  <Tab>
    ```bash
    npm install wasmer
    ```
  </Tab>
  <Tab>
    ```bash
    pip install wasmer
    ```
  </Tab>
</Tabs>

### Run a Package

Let's say you want to run the Wasmer package [`python/python`](https://wasmer.io/python/python) and execute the `-c "print(1+1)"` to print a result.

<Tabs items={['CLI', 'JS', 'Python']}>
  <Tab>
    ```bash /python\/python/
    wasmer run python/python -- -c "print(1+1)"
    ```
  </Tab>
  <Tab>

    ```js /python\/python/
    import Wasmer from "wasmer";

    let python = new Wasmer("python/python");
    let runner = python.entrypoint.run('-c "print(1+1)"');
    console.log(python.stdout);
    ```

  </Tab>
  <Tab>
    ```python /python\/python/
    from wasmer import Wasmer

    python = Wasmer("python/python")
    runner = python.entrypoint.run('-c "print(1+1)"')
    print(python.stdout)
    ```

  </Tab>
</Tabs>

</Steps>

<Callout type="info">
  You can also publish a package to the registry, so it can be used everywhere.
  [Learn how](/registry/publish).
</Callout>
