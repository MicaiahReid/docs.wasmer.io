### Deploy a package

Creates an app based on an existing package on the [registry](../registry).

Here we use <a href="https://wasmer.io/wasmer/hello-server" target="_blank">wasmer.io/wasmer/hello-server</a>,
a simple web server with a stub template page.

```bash
wasmer app create wasmer/hello-server
```

This will create an `app.yaml` file with the following contents

```yaml copy filename="app.yaml"
kind: wasmer.io/App.v1
name: <your-username>-hello-world
package: wasmer/hello-server
```

To redeploy after making changes, like upgrading the package, simply run
`wasmer deploy` in a directory with an `app.yaml` file.
