# Install Wasmer

import { Callout, Tabs, Tab } from "nextra-theme-docs";

Wasmer allows you to run applications either **Standalone** (_via the CLI_) or **Embedded** within [other languages](./#wasmer-language-integrations) such as C/C++, Rust, Python, Go, PHP, Ruby...

<Tabs items={['CLI' /*, 'JS', 'Python' */]}>
  <Tab>
    Unix (macOS, Linux):

    ```bash
    curl https://get.wasmer.io -sSfL | sh
    ```

    PowerShell (Windows):

    ```bash
    iwr https://win.wasmer.io -useb | iex
    ```

    <Callout type="info">
        Other installation methods available here: https://github.com/wasmerio/wasmer-install
    </Callout>

  </Tab>
