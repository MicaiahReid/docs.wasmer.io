import { Callout } from "nextra-theme-docs";

Wasmer allows you to run applications either **Standalone** (*via the CLI*) or **Embedded** within [other languages](./#wasmer-language-integrations) such as C/C++, Rust, Python, Go, PHP, Ruby...

  <SyncedTabs.Tab>
    Unix (macOS, Linux):

    ```bash
    curl https://get.wasmer.io -sSfL | sh
    ```

    PowerShell (Windows):

    ```bash
    iwr https://win.wasmer.io -useb | iex
    ```

        Other installation methods available here: https://github.com/wasmerio/wasmer-install
