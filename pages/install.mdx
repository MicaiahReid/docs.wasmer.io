# Install Wasmer

import { Callout } from "nextra-theme-docs";
import { SyncedTabs } from "nextra-synced-tabs";

Wasmer allows you to run applications either **Standalone** (_via the CLI_) or **Embedded** within [other languages](./#wasmer-language-integrations) such as C/C++, Rust, Python, Go, PHP, Ruby...

<SyncedTabs items={['CLI' /*, 'JS', 'Python' */]}>
  <SyncedTabs.Tab>
    Unix (macOS, Linux):

    ```bash
    curl https://get.wasmer.io -sSfL | sh
    ```

    Or, to install a specific version:

    ```bash /4\.0\.0/
    curl https://get.wasmer.io -sSfL | sh -s "4.0.0"
    ```

  </Tab>

  <Tab>
    **PowerShell** (Windows):

    ```bash
    iwr https://win.wasmer.io -useb | iex
    ```

    Or, to install a specific version:

    ```bash /4\.0\.0/
    $v="4.0.0"; iwr https://win.wasmer.io -useb | iex
    ```

  </Tab>

  </SyncedTabs.Tab>
