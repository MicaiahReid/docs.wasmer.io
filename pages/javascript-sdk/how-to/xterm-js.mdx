title: Connecting a Running Instance to XTerm.js
---

The [Wasmer.sh][wasmer-sh] website is a real-world example of how the JavaScript
SDK can be used to provide a real terminal in the browser.

After reading this how-to guide, you should be able to...

- Add an [Xterm.js][xterm-js] console to a web page
- Run an instance of [`sharrattj/bash`][bash]
- Transfer input from the XTerm.js console to the instance's `stdin` stream
- Read output from the instance's `stdout` and `stderr` streams
- Identify common errors
